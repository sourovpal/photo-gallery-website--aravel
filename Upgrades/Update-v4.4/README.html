Installation

    IMPORTANT: You must update from version 4.3, if you upgrade from another version will throw an error.
  • Upload the file v4.4.zip found inside the Update-v4.4 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v4.4.zip and enter at URL https://yoursite.com/v4.4/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v4.4/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.

  • Add this new text string in your file resources / lang / en / misc.php
    • // Version 4.4
 'info_data_iptc' => 'If the image contains IPTC data, the Title and Tags will be extracted.',
 'image_and_vector_graphic' => 'Image and Vector graphic',
 'cookie_policy_banner' => 'Cookie policy banner',
 'stripe_connect_countries' => 'Stripe Connect (Countries)',
 'info_stripe_connect_countries' => 'Countries which you want to send direct payments to your members, configure in',
 'install_web_app' => 'Install Web App',

 );//---- Don't copy this, copy before closing this array in your file
    If you have multiple languages on your site you should add to each file.
