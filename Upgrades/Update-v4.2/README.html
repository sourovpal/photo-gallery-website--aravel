Installation

    IMPORTANT: You must update from version 4.1, if you upgrade from another version will throw an error.
  • Upload the file v4.2.zip found inside the Update-v4.2 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v4.2.zip and enter at URL https://yoursite.com/v4.2/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v4.2/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
  • Add this new text string in your file resources / lang / en / misc.php
    • // Version 4.2
'error_length_tags' => 'Tags must have at least 2 characters',
'default_price_photos' => 'Default price of photos',
'info_default_price_photos' => 'If you want the user to set the price, leave it at 0',
'blog' => 'Blog',
'link_blog' => 'Link to Blog',
'apply_taxes_wallet' => 'Apply taxes in Wallet',
    If you have multiple languages on your site you should add to each file.
    IMPORTANT: If you are using Stripe you must add the event checkout.session.completed
